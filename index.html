WaterIntake

WaterIntake is a Java-based application. In this application, the user is allowed to set a goal for their daily waterintake. The default is set to 2000 in millilitres. Once run, the user will then have to input how much they have taken-in today in millilitres. If the users input is the same or larger than the set goal, they will get congradulated, knowing they have reached their target. If not, the user will be greeted with a message telling them how much more water they need to intake in order to reach their goal. To find out more, visit the GitHub link below!
